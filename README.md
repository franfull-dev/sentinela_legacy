**O teu Framework de Auditoria e Defesa Ativa | Desenvolvido por @fran.codes**

## 🚀 Funcionalidades Principais

* **🔴 Red Team - Auditoria Web:** Analisa cabeçalhos de segurança e entrega o código exato para correção.
* **🔵 Blue Team - Defesa Ativa:** Monitoriza conexões em tempo real e permite neutralizar ameaças (Kill Process).
* **🌐 Network Recon:** Scanner de rede local via ARP para detetar intrusos no Wi-Fi.
* **🔍 OSINT:** Investigação de IP e Domínios com geolocalização e dados de ISP.
* **🛠️ Tool Kit:** Gerador de hashes SHA-256 e senhas mestras de alta entropia.
